## 一、改文字内容（只改 data.js）

用记事本 / VS Code 打开 `data.js`，改完保存，刷新浏览器即可。

### 1. 改店名 / 电话 / 微信 / 地址

打开 `data.js`，找到最上面的 `基本信息` 部分：

```js
shopName: "水榭年华娱乐休闲会所",   // 改店名
phone: "0751-8609686",               // 改电话
wechat: "sx8609686",                 // 改微信客服号
address: "韶关市武江区新华南路嘉乐广场水榭年华",  // 改地址
```

### 2. 改顶部卖点横幅

在 `data.js` 找到 `uspTags`，直接改文字：

```js
uspTags: [
  "🚗 10公里免费接送",      // 改这里
  "🍊 水果茶水简餐",       // 改这里
  "🎤 K歌沐足一体"         // 改这里
],
```

### 3. 改服务项目 / 价格

在 `data.js` 里找到 `projects` 部分，
每个项目下面有 `packages`，改 `name`（套餐名）和 `price`（价格）即可。

```js
{
  name: "K歌沐足",        // 项目名（可改）
  icon: "🎤",             // 图标（随便换 emoji）
  packages: [
    { name: "貂蝉拜月", price: 178, staff: [...] },  // 改套餐名和价格
    { name: "贵妃醉酒", price: 198, staff: [...] },
  ]
}
```

- **加新套餐**：复制一行 `{ name: "...", price: 000, staff: [] },` 粘进去改
- **删套餐**：把那一行整行删除

### 4. 加技师照片

**第一步**：把技师照片放进 `images/` 文件夹
- 建议命名：`staff-1.jpg`、`staff-2.jpg` ...
- 图片格式：jpg / png / webp 都可以
- 图片大小：建议压缩到 500KB 以下（用微信发给自己再保存，会自动压缩）

**第二步**：在 `data.js` 找到对应套餐的 `staff: []`，
取消注释（删掉行首的 `//`），填上信息：

```js
staff: [
  { id: "1", img: "images/staff-1.jpg", desc: "手法娴熟，服务贴心" },
  { id: "2", img: "images/staff-2.jpg", desc: "温柔细致，好评如潮" },
]
```

> - `id` = 技师编号（显示在页面上的号码）
> - `img` = 图片路径（`images/` 里的文件名，或网络图片完整网址）
> - `desc` = 技师描述文字

**暂无照片的技师**：不加这行就行，页面自动显示"技师即将到岗"。

## 二、图片上传的完整教程

### 方式 A：本地图片（简单，推荐测试用）

1. 把图片放进 `images/` 文件夹
2. 在 `data.js` 里写 `"images/文件名.jpg"`
3. 双击 `index.html` 用浏览器打开即可看到

> ⚠️ 注意：本地图片只有在你自己电脑上能看，
> 部署到网上后别人看不到本地图片，
> **部署前需要重新部署包含图片的版本**（我可以帮你搞）。

### 方式 B：图床外链（推荐，部署后也能用）

1. 打开免费图床网站，例如：
   - **https://sm.ms**（推荐，不用注册）
   - **https://imgbb.com**（不用注册）
2. 点击「Upload」上传图片
3. 上传完成后，复制「Direct Link」（直接链接）
4. 在 `data.js` 里把 `img` 的值换成这个链接，例如：
   ```js
   img: "https://i.ibb.co/xxx/staff-1.jpg"
   ```

### 环境轮播图怎么加？

和技师照片一样，在 `data.js` 找到 `envImages`，填图片路径：

```js
envImages: [
  "images/env-1.jpg",    // 第一张
  "https://i.ibb.co/xxx/env-2.jpg",  // 也可以用外链
]
```

## 三、微信分享卡片配置（客人转发时显示的内容）

当客人把你的网站链接发给微信好友时，
微信会显示一个卡片（标题 + 描述 + 图片）。

### 改标题和描述

在 `data.js` 找到 `share` 部分：

```js
share: {
  title: "水榭年华娱乐休闲会所 · 技师风采",  // 卡片标题
  desc:  "K歌沐足·轻奢SPA·养生足浴 | 10公里免费接送 | 韶关武江区",  // 卡片描述
  image: ""   // 卡片图片（见下方教程）
}
```

### 改分享图片（重要！）

微信分享卡片的**图片必须是一个完整的网址**，不能写 `images/xxx.jpg`。

**步骤**：

1. 准备一张正方形图片（建议 200×200 或 500×500 像素）
   - 可以是店铺 Logo、门头照片、或任何代表店铺的图片
2. 上传到图床（见「方式 B」），拿到完整网址
3. 做两件事：
   - **在 `data.js`** 里填 `share.image`：
     ```js
     image: "https://i.ibb.co/xxx/share.jpg"   // 换成你的图片网址
     ```
   - **在 `index.html`** 里改 `og:image` 这行（用记事本打开）：
     ```html
     <meta property="og:image" content="https://i.ibb.co/xxx/share.jpg">
     ```
     > 这步必须做！因为微信抓取链接时不会执行 JS，
     > 只会读取 HTML 里的 `og:image` 标签。
